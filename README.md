# ReactBlueprint

Modern patterns, strict conventions, and battle-tested architecture for React + TypeScript projects.

## About

ReactBlueprint is a comprehensive documentation site covering best practices for building scalable React applications. It includes guidelines for:

- Project structure and naming conventions
- Component patterns and architecture
- State management (Zustand, TanStack Query)
- TypeScript configuration and patterns
- Testing strategies
- Performance optimization
- Accessibility
- And more...

## Tech Stack

- [Next.js](https://nextjs.org/) - React framework
- [Nextra](https://nextra.site/) - Documentation theme
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [Vercel](https://vercel.com/) - Deployment

## Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```
